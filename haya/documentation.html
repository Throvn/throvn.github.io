Change Sounds:
How to change

  1. Look in the documentation which var you want to change
  2. Then change the sound by writing: 
         audio = new Audio('//path');
    • (or you change the path directly in the soundsInJs.js File)
  3. Done

Change Volume
How to change

  1. Look in the documentation of which var you want to adjust
  2. After that look in the documentation which level the volume has now
  3. Then change the sound by writing: 
         audio.volume = 0.2;
    • (or you change the path directly in the soundsInJs.js File)
  4. Done

Buttons

Information: 
    Eventlistener: Click
    Variable:      audio1
    Volume:        0.2
Code: 
//button click

var button = document.getElementsByTagName('button');
for (var i = 0; i < button.length; i++) {
  button[i].addEventListener("click", function(){
      var audio1 = new Audio('./library/btn_click.mp3');
      audio1.volume = 0.2;
      audio1.play();
  });
}

Typing sounds
You can hear the typingsounds if you write in an input e.g.

Information:

This sound can be pretty annoying sometimes, so I decided to remove it as the default...
If you want to hear the typing sounds you have to add the class: type_sou to your input.

Information: 
    Eventlistener: keydown
    Variable:      audio9
    Volume:        1
Code: 
//typing sounds

var type = document.querySelectorAll('.type_sou');
for (var i = 0; i < type.length; i++) {
  type[i].addEventListener("keydown", function(){
    var rand = Math.floor(Math.random() * (5 - 1 + 1)) + 1;
    var audio9 = new Audio('./library/typing/typing'+rand+'.mp3');
    audio9.play();
  });
}
